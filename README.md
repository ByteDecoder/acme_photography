# acme_photography

CSS3 playground

## Instructions

```bash
yarn install
yarn dev
```

## Production build

```bash
yarn build
firebase login
firebase deploy
```

More info about firebase deploy in: <https://medium.com/google-developer-experts/deploy-your-app-to-firebase-in-seconds-b3a9a37dff47>
